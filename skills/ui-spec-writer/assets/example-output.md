在會員模組底下新增等級禮設定頁面類別，子頁面有 "入會禮設定", "升等禮設定", "等級獨享禮設定"

### 入會禮設定頁

以設定區塊的方式條列呈現各個等級，每個等級區塊都有編輯按鈕，點擊"編輯"按鈕後前往 "編輯 {會員等級名稱} 入會禮設定" 頁
每個等級設定區塊中的內容:

```
+---------------------------------------------------------------------------------+
|                                                                                 |
|  | {等級X名稱} 入會禮                                            [ 編輯 ]  |
|                                                                                 |
|   啟用狀態                啟用                                                   |
|                                                                                 |
|   發放內容                                                                      |
|   堂票                    {堂票名稱} x {數量} {單位}                            |
|                           {堂票名稱} x {數量} {單位}                            |
|                                                                                 |
|   票券                    {票券名稱} x {數量} 張                                |
|                           {票券名稱} x {數量} 張                                |
|                                                                                 |
|   點數                    {點數名稱} x {數量} {單位}                            |
|                           {點數名稱} x {數量} {單位}                            |
|                                                                                 |
|   回饋金                  {數量}                                                |
|                                                                                 |
|   圖文訊息                已設定                                                |
|                                                                                 |
+---------------------------------------------------------------------------------+
```

發放內容 如果還沒有設定的話顯示 "尚未設定"。

#### "編輯 {會員等級名稱} 入會禮設定" 頁

起初會有 "發放設定" 區塊，區塊內有:
啟用狀態(開關)
發放種類(下拉選單, 多選, 必填)
placeholder: 請選擇發放種類
發放種類的選項有: 堂票, 票券, 點數, 回饋金, 圖文訊息
選了哪些種類選項，底下就會出現對應的種類設定區塊

堂票區塊:

- 發放堂票 (堂票進階選擇器)
- 發放數量 (數字輸入 + {單位})
- 新增發放堂票按鈕 (多新增"發放堂票"、"發放數量"區塊，第二組開始都會有移除按鈕)

票券:

- 發放票券 (票券進階選擇器)
- 發放數量 (數字輸入 + {單位})
- 新增發放票券按鈕 (多新增"發放票券"、"發放數量"區塊，第二組開始都會有移除按鈕)

點數:

- 發放點數 (下拉選單)
- 發放數量 (數字輸入 + {單位})
- 新增發放點數按鈕 (多新增"發放點數"、"發放數量"區塊，第二組開始都會有移除按鈕)

回饋金:

- 發放數量 (數字輸入 + 元)

圖文訊息:

- 訊息 (圖文訊息組件)
- 新增按鈕

### 升等禮設定頁

與 `入會禮設定頁` 形式一致，但在每個等級區塊中的 "啟用狀態" 下多一個 "領取次數":

```
|   領取次數                    單一會員可重複領取 or 單一會員只能領取 1 次                      |
```

點擊各等級區塊的 "編輯" 按鈕後前往 "編輯 {會員等級名稱} 升等禮設定" 頁

#### "編輯 {會員等級名稱} 升等禮設定" 頁

與 "編輯 {會員等級名稱} 入會禮設定" 頁形式一致，但 "發放設定" 區塊的 "發放種類(必填)" 下方多一個 "領取次數規則(必填, 下拉選單, 單選)"，選項: "單一會員可重複領取", "單一會員只能領取 1 次"

### 等級獨享禮設定頁

與 `入會禮設定頁` 形式基本上一致
點擊各等級區塊的 "編輯" 按鈕後前往 "編輯 {會員等級名稱} 等級獨享禮設定" 頁

#### "編輯 {會員等級名稱} 等級獨享禮設定" 頁

與 "編輯 {會員等級名稱} 入會禮設定" 頁形式一致

### 等級禮發放紀錄頁

頁面 title: 等級禮發放紀錄
列表過濾元件，包含:

- 會員選擇 (MemberSearch 組件)
- 等級禮類型 (BaseDataSelect, 單選, placeholder: 請選擇等級禮類型)
- 發放狀態 (BaseDataSelect, 單選, placeholder: 請選擇發放狀態)
- 開始時間與結束時間 (daterange picker)

過濾元件布局:

```
會員選擇
等級禮類型  發放狀態
開始時間與結束時間
```

列表欄位:

- 會員姓名
- 會員電話
- 等級禮類型
- 發送狀態 (tag)
  - 狀態: 待處理(warning), 發放中(warning), 已發放(action), 部分失敗(danger), 發放失敗(danger)
- 建立時間 (yyyy/mm/dd HH:mm:ss)
- 發送完成時間 (yyyy/mm/dd HH:mm:ss)
- 檢視
  - 檢視按鈕 (點擊後開啟 "drawer: 等級禮發放紀錄詳情")

#### drawer: 等級禮發放紀錄詳情

上方基本資訊欄位：

- 操作 (例如：回收獎勵 連結)
- 會員姓名
- 會員電話
- 等級禮類型
- 觸發來源
- 原會員等級
- 異動後會員等級
- 發送狀態 (tag)
- 建立時間 (yyyy/mm/dd HH:mm)
- 發送完成時間 (yyyy/mm/dd HH:mm 或 "-")

區塊：會員禮發放明細
以卡片形式呈現各項發放項目內容，欄位包含：

- 類型
- 名稱
- 發放數量
- 回收數量 (註：發送狀態為「已回收」或「回收失敗」才會有此欄位)
- 無法回收數量 (註：發送狀態為「已回收」或「回收失敗」才會有此欄位)
- 發送狀態 (tag)
  - **已回收**：發放數量 ≤ 回收數量
  - **回收失敗**：回收數量 = 0
- 執行結果 (註：僅在發送狀態為「回收失敗」時顯示，例如："自動回收失敗：error message")
- 備註 (註：僅在發送狀態為「回收失敗」時顯示)
- 實際發放時間 (yyyy/mm/dd HH:mm)
- 會員禮回收時間 (yyyy/mm/dd HH:mm)

底部操作區：

- 返回 (按鈕)
